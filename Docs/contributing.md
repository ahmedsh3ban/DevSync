# 🤝 دليل المساهمة في DevSync (Contributing Guide)

سعداء جداً بقرارك للمساهمة معنا! لضمان سير العمل بشكل منظم، يرجى اتباع الخطوات التالية للمشاركة في أي مشروع:

## 🚀 خطوات البدء
1. **Fork:** قم بعمل `Fork` للمستودع (Repository) على حسابك الشخصي.
2. **Clone:** قم بتحميل المشروع على جهازك باستخدام الأمر:
   `git clone https://github.com/your-username/DevSync.git`
3. **Branch:** أنشئ فرعاً جديداً للمهمة التي ستعمل عليها:
   `git checkout -b feature/your-feature-name`

## 💻 أثناء العمل
* تأكد من اتباع **قواعد الفريق (rules.md)** الموجودة في مجلد `Docs`.
* قم بكتابة كود نظيف ومنظم، واستخدم مكونات ريأكت (React Components) إذا كان المشروع يعتمد عليها.
* تأكد من اختبار الكود الخاص بك محلياً قبل الرفع.

## 📤 إرسال المساهمة
1. قم بإضافة التغييرات: `git add .`
2. قم بعمل كوميت بوصف واضح: `git commit -m "feat: description of your work"`
3. ارفع التغييرات إلى حسابك: `git push origin feature/your-feature-name`
4. اذهب إلى المستودع الأصلي وقم بفتح **Pull Request (PR)**.

## ✅ مراجعة الكود
* سيقوم مسؤول الفريق (Team Leader) بمراجعة الكود الخاص بك.
* قد يتم طلب بعض التعديلات قبل قبول الدمج (Merge).
* بمجرد قبول الـ PR، سيصبح كودك جزءاً من المشروع الأساسي!
